# alipine-blog

## Description

友人のために作成した登山ブログ

## DEMO

https://alpine-blog.vercel.app/

## Components

- Next.js
- microCMS (Headless CMS)
- Vercel (Hosting)
- tailwindcss
- Storybook

## Usage

### Setup

```
$ npm install
```

### Build

```
$ npm run build
```

### Storybook

```
$ npm run storybook
```
